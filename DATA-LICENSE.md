# Data license and attribution

This repository contains a vetted atlas dataset and derived artifacts used to power the site.

## Vetted atlas source

The atlas build pipeline now uses a single local source file:

- `data/raw/participatory_ai_atlas_expanded_2026-03-23.csv`

Generated JSON, GeoJSON, and release files in this repository are transformations of that bundled source dataset and should be used with the same attribution expectations as the source CSV.

## What this file covers

- repository code: MIT
- bundled raw and derived data: attributed to the source atlas dataset and subject to its stated license or usage conditions
